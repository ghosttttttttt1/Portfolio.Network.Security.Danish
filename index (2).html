// job responsibilities

Monitor network traffic, logs, and security events to detect suspicious or malicious activity
Perform vulnerability assessments and network scanning to identify weaknesses before attackers do
Analyse packet captures (using Wireshark) to understand the nature and source of threats
Implement and maintain firewalls, IDS/IPS, VPNs, and network access controls
Investigate security incidents, document findings, and escalate critical cases
Maintain network security policies and ensure compliance with security standards
Produce security reports and recommendations for management

Every government agency, bank, telco, and enterprise needs a dedicated person watching over their network 24/7. The Network Security Analyst is that person — the first line of defence against digital threats. With Malaysia's push toward a digital economy and the growing number of connected devices, demand for this role continues to surge, with a clear career path toward roles such as Security Architect, Penetration Tester, or Cloud Security Engineer.
